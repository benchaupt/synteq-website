# Design System Notes

Visual rules that aren't captured in code but must be followed. If something isn't listed here, check existing components before guessing.

---

## Corners

**No rounded corners.** The site uses sharp/square corners everywhere — cards, containers, borders, buttons. Do not add `rounded-*` classes unless explicitly told to.

---

## Borders

- Use `border-cream` for all dividers and card borders
- Thin 1px borders only — no thick or double borders
- Grid dividers use individual cell borders, not gap-based spacing
